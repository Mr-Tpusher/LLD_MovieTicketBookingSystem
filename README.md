### Requirements:
1. Users should be able to see all the Cinemas in their city
2. Every cinema will have multiple halls.
3. After clicking on one of the cinemas, they should be shown all the shows available for that day in that cinema.
4. After selecting a show, users should be able to select seats and make the booking
   a. seats are going to be of 3 types {normal, gold, platinum}
5. Users can also search for a specific movie and all the cinemas having that movie's shows should be displayed
6. User should be able to cancel a booking
7. User should receive booking confirmation on email and sms
